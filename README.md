Aplikasi dibuat dengan menggunakan 
  1. Framework Django, Flask
  2. Event bus RabbitMQ
  3. SQLITE3

Buat 2 buah venv untuk setiap aplikasi Django dan Flask, kemudian pip install -r requirements.txt pada setiap venv
Buat akun pada https://www.cloudamqp.com, untuk membuat akun RabbitMQ. Pilih akun yang FREE.
Pada AMQ Detail Copy baris URL kemudian paste pada producer.py dan consumer.py
